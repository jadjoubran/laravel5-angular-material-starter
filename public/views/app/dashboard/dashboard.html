Laravel 5

The Laravel 5.1 that you loved, comes pre-configured to run with AngularJS:

  • JWT Authentication
  • A view (resources/index.blade.php) that loads all the assets you need (CSS and JS) & sets up angular ui-router
  • Support for CSRF token when using the API
  • Debugbar for Laravel 5: It's that bar you see at the bottom. Very useful for debugging the API
  • Sample API endpoints
  • Elixir configuration for Angular, jshint, bower and views.
  • Optimized deployment script.
  • API response macro

AngularJS

AngularJS comes ready to communicate with the Laravel 5 API that you're building. It actually comes with a few sample API endpoints!
This is what you have out of the box:

  • A flexible and powerful router for Angular: ui-router
  • Restangular makes communicating with the API a breeze!
  • Support for CSRF token via Restangular
  • Sample UI router with multiple views (main & footer)
  • UI Router sample authentication
  • ngstorage module which simplifies access to localStorage.. The Angular way!
  • Angular Material Icons with SVG morpheus

Angular Material

Material Design as a UI library! Chosen instead of bootstrap in order to avoid injecting jQuery with Angular.
Here's what you'll get:

  • Sample Landing and Dashboard pages design with Material Design
  • Custom Material Design theme so that you can easily create your own theme
  • Sample Dialog, Toast notifications & Tabs
  • More coming soon!
