dingo/API

 

$api->version('v1', function ($api) {
    $api->get('data', 'App\Http\Controllers\WelcomeController@getSample');
});

dingo/api package is meant to provide you, with a set of tools to help you easily and quickly build your own API. dingo/api works out of the box in Laravel angular material starter.
Feel free to check their wiki to customize it. dingo/api is configured to work harmoniously with JWT-auth.

Restangular

 

Restangular.all('data').doGET().then(function(response){
    console.log(response.data);
});

Restangular is an AngularJS service to handle Rest API Restful Resources properly and easily.

API Response Macro

 

//return your data using the api response macro
return response()->api($data);

ResponseMacroServiceProvider returns your data inside the data object.

You can customize it to include other properties.

Also route versioning and Accept header are configured for you in angular/config/restangular.js, to match with Dingo/API.
